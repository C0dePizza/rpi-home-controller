RPi Home Controller

Material Design based web-app that controls your home.

View on GitHub Download .zip Download .tar.gz

Overview

The RPi Home Controller is a material design interface for the GPIO of your Raspberry Pi, allowing easy manipulation of relays.

With this you can control lighting, your HVAC system, and anything else you can plug into a power outlet (eg. toaster, TV, window unit air conditioners).

Useful links

Super easy to use 110v 'relay' here.

Or if you want to do it yourself, this Instructable is a great way to wire up your home.

Features

  • Supports the DS18B20 temperature sensor and can control conventional HVAC systems as well as graph your temperature history using Plotly
  • Can scan your local network for your mobile phone(s) and can shut off lights and air conditioning when nobody is home
  • Aesthetically appealing web interface
  • Synchronization across web clients
  • An additional TCP socket server (for easy integration with third party applications)
  • Fully customizable display

Usage

# ./main.py -h

usage: main.py [-h] [--tcp] [--http] [--ws] [--ping] [--thermostat]

Launcher for RPi Home Controller.

optional arguments:
  -h, --help    show this help message and exit
  --tcp         Enable TCP server
  --http        Enable HTTP server
  --ws          Enable WebSocket server
  --ping        Enable Ping worker
  --thermostat  Enable Thermostat worker
